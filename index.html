Bastion Sensor Package

Electromechanical Engineer

  • Check out the project

The Bastion Sensor Package was my senior design project that I worked on with a team during my graduating year at the University of Colorado Boulder. This project was sponsored by Sandia National Laboratories, and in this project, we designed and fabricated a rapidly-deployable sensor package that would scan a 10-meter radius and alert the user if a vehicle or person was detected.

As an electromechanical engineer, I assisted the other two electromechanical engineers in choosing the best sensors for the project, designing the PCBs, soldering, and wiring the final assembly together. My biggest contribution to the project was implementing the object segmentation model, so the sensor package could determine, with at least a 90% accuracy, if an alert was a person, vehicle, or false alarm.
